# procbins
compresses all process binaries into a zip file

This tool is intended to be used for forensic triage.

## Usage
```
Usage:
  procbins ZIPFILE

compresses all process binaries into a zip file

Positional arguments:
  zipfile               name of the destination zip file

Optional arguments:
  -h,--help             Show this help message and exit
```
